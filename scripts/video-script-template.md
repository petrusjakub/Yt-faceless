## Script Quality Checklist

Before recording/generating narration:

- [ ] Hook grabs attention in first 5 seconds
- [ ] Clear promise of what viewer will learn
- [ ] All statistics verified and accurate
- [ ] Primary keyword appears 3-5 times naturally
- [ ] Each section has a clear point and example
- [ ] Transitions between sections feel natural
- [ ] Actionable takeaway is specific (not generic)
- [ ] CTA feels natural, not forced
- [ ] Total word count matches target length
- [ ] No jargon used without explanation
- [ ] Tone is confident but not arrogant
- [ ] No misleading claims or guarantees
