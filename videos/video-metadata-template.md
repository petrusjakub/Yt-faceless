## Upload Checklist

### Pre-Upload
- [ ] Video exported in 1080p or higher (MP4 format)
- [ ] Audio levels consistent throughout (no clipping, no silence gaps)
- [ ] All statistics displayed on-screen are accurate and verified
- [ ] No copyrighted music or footage used
- [ ] End screen compatible (last 20 seconds have space for elements)
- [ ] Video length is between 8-15 minutes

### Thumbnail
- [ ] Primary thumbnail (Version A) created at 1280x720
- [ ] A/B thumbnail (Version B) created for testing
- [ ] Thumbnail is readable at small/mobile size
- [ ] Text has maximum 5 words
- [ ] Colors match brand (navy + gold + white)

### Metadata & SEO
- [ ] Title is under 60 characters
- [ ] Primary keyword is in first half of title
- [ ] Description first 2 lines are compelling (visible in search)
- [ ] Timestamps added in description
- [ ] 15-20 tags added (primary keyword first)
- [ ] Category set to "Education"
- [ ] Language set to English

### Settings
- [ ] End screen elements configured (subscribe + videos)
- [ ] Cards added at relevant timestamps (if applicable)
- [ ] Video added to appropriate playlist
- [ ] Comments enabled
- [ ] Not set as "made for kids"
- [ ] Scheduled for optimal time

### Post-Upload (within 24 hours)
- [ ] Pinned comment posted
- [ ] Community post teasing the video published
- [ ] Create 1-2 YouTube Shorts from video highlights
- [ ] Share on social media (Twitter, relevant subreddits)
- [ ] Reply to first 10 comments
- [ ] Verify video appears in playlist correctly

### Performance Check (48 hours after publish)
- [ ] Check CTR (target: above 5%)
