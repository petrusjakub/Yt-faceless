## Content Calendar Notes

### Video Priority Scoring

Each video was selected based on:
- **Search volume** (monthly searches for target keyword)
- **Competition level** (how many similar videos exist)
- **Revenue potential** (CPM for that topic category)
- **Audience match** (alignment with target persona)

### Content Repurposing Plan

Each long-form video should be repurposed into:
1. **1-2 YouTube Shorts** (60-second key takeaways)
2. **Twitter/X thread** (summarize main points)
3. **Pinterest pin** (infographic version of key data)
4. **Community post** (poll or discussion based on video topic)
